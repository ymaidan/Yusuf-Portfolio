# Gallery Images

Place your certificate and achievement images in this folder.

## Required Images:

Based on your resume, you should add the following images:

1. **aws-cert.jpg** - AWS Cloud Practitioner Certificate
2. **cisco-cert.jpg** - Cisco CCNA Certificate
3. **huawei-comp.jpg** - Huawei ICT Competition photo
4. **seeds-future.jpg** - Huawei Seeds For The Future certificate
5. **ryla.jpg** - Rotary Youth Leadership Award certificate
6. **uni-project.jpg** - University project (Smart Waste Management) photo
7. **first-aid.jpg** - First Aid/CPR/AED certificate
8. **toastmasters.jpg** - Toastmasters photo/certificate

## Image Guidelines:

- **Format:** JPG, PNG, or WebP
- **Resolution:** At least 800x600px recommended
- **Aspect Ratio:** 4:3 works best
- **File Size:** Keep under 2MB for faster loading

## Note:

If an image is missing, the gallery will show a placeholder with the title text. You can add images gradually.
